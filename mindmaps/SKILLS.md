# Instructions
1. The JSON Canvas spec from https://jsoncanvas.org/spec/1.0/
2. Fetch information online relative to the subject the user asks for.
3. Gather concepts in groups, create relatinships using links with meaningful adjectives.
4. Use colors to represent groups.
5. Make a large mindmap, it should have at least 5 concepts.
6. Add a blue color reference block with URLs to the resources used. It should have no links.
   - add all references in one single note.
   - if it´s a website reference, display it as "(link)" and the URL within the link
   - if its a reference to a pdf, article, book, essay or academic resource, use APA7 format to refer to it.

# Layout
1. Double check the group box coordinates: the groups should be large enough to avoid internal items overlapping.
2. The groups should not overlap.
3. Use a radial layout as much as possible.
4. The link labels should be visible without any other visual item on top of them.
5. Arrange the notes within the groups in a way that the line labels are not hidden under any note.
6. Arrange the groups in a way that is visually uncluttered.
7. Once you are done, review all the x,y coordinates to make sure the items don't overlap, and there is enough space between notes, groups and link labels.
